Elevate / Overview

Elevate is a lightweight CSS framework for building gamified learning platforms. It provides a base layer of typography, layout utilities, including advanced components for lessons, quests, mastery, progress, xp and leaderboards.

Base

Professional surfaces

Tokens, typography, layout utilities, cards, forms, and navigation for serious learning products.

XP

Gamified learning

Progress bars, mastery rings, level pills, badges support learner momentum.

Portable

Static friendly

Designed to work seamlessly in static hosting environments

Getting Started

To get started, include the Elevate CSS and JS bundle from elevate.openlib.in.

CSS <link rel="stylesheet" href="https://elevate.openlib.in/elevate.min.css">
JS <script src="https://elevate.openlib.in/elevate.min.js"></script>
