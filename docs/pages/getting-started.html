Elevate Theme

A professional dark UI theme for gamified fullstack learning platforms, with components for lessons, quests, mastery, progress, and learner operations.

Install from GitHub Pages

Use the minified bundle when the repository is published at vstechapps.github.io/elevate.

<link rel="stylesheet" href="https://vstechapps.github.io/elevate/dist/elevate.min.css">
Build the CSS bundle package.json

Run the build script before publishing so dist/elevate.min.css is available to the docs and external consumers.

npm run build:css npm run build 
{
  "scripts": {
    "build": "npm run build:css",
    "build:css": "node scripts/build-css.mjs"
  }
}
