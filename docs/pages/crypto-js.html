Client-side usage

Use the bundled helper from the browser after the script is loaded.

const secret = "my-secret";

const encrypted = await window.Crypto.encrypt("Hello world", secret);
const decrypted = await window.Crypto.decrypt(encrypted, secret);

const encryptedJson = await window.Crypto.encjson(JSON.stringify({ name: "Ada" }), secret);
const decryptedJson = await window.Crypto.decjson(encryptedJson, secret);

For JSON payloads, the helper encrypts both the object keys and the values.
